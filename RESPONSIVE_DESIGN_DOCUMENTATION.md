# Responsive Design Documentation

## Overview
This project was optimized for mobile, tablet, and desktop screens.

## Responsive Techniques Used

### Media Queries
- Mobile: max-width 768px
- Tablet: max-width 1024px
- Desktop: above 1024px

### Layout Changes
- Desktop: 3-column layout
- Tablet: 2-column layout
- Mobile: Single-column layout

### Typography
- Used rem units for scalable text.
- Reduced heading sizes on mobile.

### Navigation
- Navigation stacks vertically on smaller screens.

### Images
- Images use max-width: 100% to prevent overflow.

## Testing
Tested using:
- Chrome DevTools
- Mobile View
- Tablet View
- Desktop View
